---

## 1. 设计原则

### 1.1 视觉风格

- **参考标杆**：Microsoft To Do 的交互和布局模式
- **融入项目风格**：复用 GuYanTools 现有的主题系统、字体配置、圆角/阴影风格
- **暗色/亮色适配**：所有组件适配现有的主题切换系统
- **微交互动画**：任务完成、添加、删除等操作均配有流畅的过渡动画

### 1.2 布局原则

- **左右分栏**：左侧侧边导航 + 右侧内容区
- **详情面板**：点击任务后右侧滑出详情面板（三栏布局）
- **响应式**：窗口缩小时侧边栏可折叠，详情面板自适应

#### 交互细节

| 操作                 | 行为                                              |
| -------------------- | ------------------------------------------------- |

| 点击用户列表项       | 切换右侧内容区到该列表视图                       |

### 3.2 TodoItem（任务条目）

#### 布局结构

```
┌──────────────────────────────────────────────────────────┐
│ ○  任务标题                            ☆  📅 3/25      │
│    步骤 2/5                                              │
└──────────────────────────────────────────────────────────┘
```

| 元素             | 位置   | 说明                                            |
| ---------------- | ------ | ----------------------------------------------- |
| 完成复选框 `○`   | 左侧   | 圆形，点击切换完成状态，完成后变为 `✓` + 填充色 |
| 任务标题         | 中间   | 完成后添加删除线和降低透明度                     |
| 步骤进度         | 标题下 | 仅在有步骤时显示，如 "步骤 2/5"                 |
| 重要星标 `☆`     | 右侧   | 点击切换重要状态，重要时变为实心 `★`             |
| 截止日期标签     | 右侧   | 过期红色，今日橙色，未来灰色                     |
| 提醒图标 🔔      | 右侧   | 仅在设置了提醒时显示                             |
| 重复图标 🔄      | 右侧   | 仅在设置了重复时显示                             |

#### 交互动画

| 状态       | 动画效果                                |
| ---------- | --------------------------------------- |
| 新增       | 从上方滑入 + 淡入，duration: 300ms      |
| 完成       | 复选框弹性缩放 → 标题删除线从左到右扫过  |
| 删除       | 向右滑出 + 淡出，duration: 250ms        |
| 拖拽排序   | 被拖拽项浮起阴影，目标位置显示插入指示线 |
| hover      | 微微抬起 + 背景色变化                   |

### 5.1 颜色体系

#### 亮色主题

| 用途             | 色值      | 说明                         |

| 重要色           | `#E8553D` | 星标填充色                   |

| 分割线           | `#E5E7EB` | 各区域分隔                   |

#### 暗色主题

| 用途             | 色值      | 说明                         |
| ---------------- | --------- | ---------------------------- |
| 主色调           | `#6BA3D6` | 适配暗色对比度               |
| 完成色           | `#5B9BD5` | 保持一致                     |
| 背景色           | `#1E1E1E` | 内容区背景                   |

- 标题：基于 `baseSize` 的 1.2x

- 辅助信息（日期、步骤进度）：`baseSize` 的 0.85x
- 遵循项目的统一字体 (`useAppConfigStore` 中的字体配置)

## 6. 关键交互流程

### 6.1 添加任务流程

```mermaid
sequenceDiagram
    actor User as 用户
    participant QA as QuickAdd
    participant Store as todoStore
    participant List as 任务列表

    User->>QA: 点击输入框，输入标题
    User->>QA: 按下 Enter
    QA->>Store: addTodo(title, listId)
    Store-->>QA: 清空输入框
    Store-->>List: 插入新 TodoItem (动画)
    List-->>User: 新任务从底部展开出现
```

### 6.2 完成任务流程

```mermaid
sequenceDiagram
    actor User as 用户
    participant Item as TodoItem
    participant Store as todoStore
    participant List as 任务列表

    User->>Item: 点击复选框
    Item->>Item: 复选框弹性动画 ✓
    Item->>Item: 标题删除线扫过动画
    Item->>Store: completeTodo(id)
    Store-->>List: 任务移至已完成区域 (300ms 延迟)
    alt 是重复任务
        Store-->>List: 新周期任务插入到列表中
    end
```

### 6.3 打开详情流程

```mermaid
sequenceDiagram
    actor User as 用户
    participant Item as TodoItem
    participant Detail as TodoDetail面板

    User->>Item: 点击任务标题
    Item->>Detail: 面板从右侧滑出 (300ms ease)
    Detail-->>User: 展示任务所有详细信息
    User->>Detail: 编辑任意字段
    Detail->>Detail: 每次编辑自动保存 (debounce 500ms)
    User->>Detail: 点击关闭 ✕
    Detail-->>Detail: 面板向右滑出收起
```

## 9. 无障碍 (Accessibility)

- 所有交互元素支持 Tab 焦点导航
- 复选框、按钮等有 `aria-label` 描述
- 颜色对比度满足 WCAG AA 标准
- 拖拽操作有替代的键盘操作方式
