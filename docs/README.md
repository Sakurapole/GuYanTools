# Documentation Index

项目中的正式 Markdown 文档已统一收拢到 `docs/` 下，并按主题拆分目录：

- `plans/`：实施方案与设计草案
- `desktop/`：Electron 桌面端相关说明
- `core/`：`multi_platform_core` 核心库文档
- `mobile/`：Flutter 移动端与资源说明
- `components/`：前端组件与图标系统文档
- `misc/`：临时草稿或未归类文档

当前目录结构：

```text
docs/
├── README.md
├── plans/
├── desktop/
├── core/
├── mobile/
├── components/
└── misc/
```

为了兼顾可发现性，少数原目录下仍保留了非常薄的 `README.md` 跳转页；正式内容以 `docs/` 中的版本为准。
