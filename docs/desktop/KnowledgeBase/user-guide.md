# 知识库用户指南

## 1. 入口

- 从应用底部栏进入“知识库”。
- 首次进入会自动创建默认知识库、默认空间和“快速收集箱”。
- 顶栏可搜索当前库、导入文件、新建空间、新建文件夹、新建 Markdown 页面、块页面或画布页面。

## 2. 写作与保存

- Markdown 页面支持源码编辑、工具栏插入图片/附件、插入 `[[页面名]]` 双链。
- Markdown 页面支持分屏即时预览、预览模式、Focus Mode、Typewriter Mode 和编辑器局部主题。
- Frontmatter 面板可编辑页面开头的 `--- ... ---` 属性块，正文预览会隐藏该属性块。
- 工具栏可插入表格、Callout、数学块和 Mermaid 代码块；当前数学和 Mermaid 以安全预览容器展示，不执行外部渲染脚本。
- 导出入口支持 Markdown、HTML 下载，以及通过系统打印流程输出 PDF。
- 块页面支持正文、标题、列表、任务、代码、引用、提示、图片和附件块。
- 画布页面支持文本框、图片、矩形标注、箭头和自由线条，适合整理截图、流程和资料拼贴。
- 画布页面可粘贴截图或插入图片；图片会先进入知识库附件，再写入画布元素。
- 画布元素可记录标题、页面引用和 Todo ID；保存后这些文本会进入搜索索引，也可从元素属性面板跳转到目标页面或 Todo。
- 画布页面支持 SVG/PNG 导出；复杂图片画布建议保留 SVG 作为兜底。
- 页面编辑会延迟自动保存；切换页面、窗口关闭和页面隐藏时会尝试保存脏草稿。
- 如果关闭时仍存在未保存草稿，应用会触发离开保护。

## 3. 文件导入

- 点击“导入”选择 Markdown、文本、PDF、Word、PPTX、Excel、图片等文件。
- 导入会把原文件复制到知识库附件目录，并创建可搜索的导入文档页面。
- 超过设置页“单文件导入上限”的文件会被跳过。
- 抽取失败不会删除原文件，失败原因会出现在跳过提示或最近索引任务中。

## 4. 速记

- 可通过设置页“快捷键”区域配置“知识库速记”和“剪贴板转速记”系统快捷键。
- 速记默认进入快速收集箱。
- 速记可转为知识库页面或 Todo；转换后保留来源关系。

## 5. 标签、双链和 Todo

- Inspector 的标签页可创建标签、绑定当前页面/文档/速记/附件。
- Markdown 中输入或通过工具栏插入 `[[页面名]]`，保存后会生成出链和反链。
- 未创建页面的双链会显示为缺失页面，可从反链面板创建。
- 块页面中的任务块可转为 Todo，Todo 详情会显示知识库来源并可跳回。

## 6. 设置

在设置页的“知识库”页签中可以配置：

- 默认知识库：未指定库的导入和附件粘贴会使用该库。
- 附件目录策略：默认写入应用数据目录，也可选择自定义目录。
- LibreOffice 路径：为后续 Office 高保真转换预留。
- 全文索引：关闭后导入只保存文件和元数据。
- 单文件导入上限：保护应用避免被超大文件拖慢。
- 预览缓存保留和手动清理：保留天数用于自动清理过期预览缓存，`0` 表示只手动清理；手动清理只删除预览缓存，不删除原始附件。

## 7. 故障处理

- 导入失败：查看右侧 Inspector 的最近索引任务，失败或取消任务可点击“重试”；“取消”只把任务状态置为已取消，不保证中断已经开始的同步抽取。
- 文件太大：调整设置页“单文件导入上限”，或拆分文件后再导入。
- 搜索不到新文件：确认全文索引已开启；若任务失败，点击重试。
- 附件打不开：确认原始附件仍在知识库附件目录中，必要时在系统中显示定位文件。
