# 知识库 UI 设计

> 版本：0.1
> 日期：2026-05-27
> 文档状态：草案

## 1. 页面结构

知识库采用工具型三栏布局，不做营销式首页。

```text
顶部工具栏
├─ 当前库/标题
├─ 搜索框
└─ 新建页面 / 导入 / 速记

左侧资料树
├─ 快速收集箱
├─ 最近编辑
├─ 收藏
└─ 知识库/空间/文件夹/页面

中间工作区
└─ 页面编辑器或文档预览

右侧 Inspector
├─ 目录
├─ 反链
├─ 标签
├─ 附件
├─ Todo
└─ AI
```

## 2. V0.1 页面

V0.1 只提供空状态骨架：

- 顶部工具栏。
- 左侧资料树占位。
- 中间空状态。
- 右侧 Inspector 占位。

按钮处于 disabled 状态，避免用户误以为功能已经可用。

## 3. V0.3 页面

V0.3 接入真实数据：

- 左侧树展示默认库、默认空间和快速收集箱。
- 支持新建空间、文件夹、Markdown 页面。
- 支持选中页面后加载中间区。
- 右侧 Inspector 展示页面基础属性。

## 4. 交互原则

- 资料树密度要高，适合长期扫描。
- 页面编辑区优先给内容留空间。
- 右侧 Inspector 可折叠。
- 搜索入口始终在顶部可见。
- 速记入口始终可一键触达。

## 5. 空状态文案

空状态只说明当前状态，不解释完整功能：

- 主标题：等待创建第一条知识内容。
- 副文本：库、空间、页面、附件、标签和反链结构已预留。

后续真实功能上线后，空状态改为直接动作：

- 新建 Markdown 页面。
- 导入文件。
- 打开速记。

## 6. 响应式

- 宽屏：三栏。
- 中等宽度：隐藏右侧 Inspector。
- 窄屏：隐藏左侧资料树，只保留中间区。

桌面端以常规窗口尺寸为主，移动适配不是首要目标。

## 7. 组件和主题约束

- 操作按钮、输入框、字段、弹窗、菜单、Tab、图标按钮优先使用主应用已有通用 UI 组件，不在知识库页面内重复造基础控件。
- 浏览器原生 `prompt()`、`confirm()`、`alert()` 不适用于 Electron 当前渲染环境；需要输入文本时使用全局 `TextPromptDialog`，需要危险确认时使用全局确认弹窗。
- 如果发现缺少可复用基础控件，且该控件不属于知识库专有业务，应放入 `desktop/src/windows/main/components/ui/` 和 `desktop/src/windows/main/composables/`，再由知识库调用。
- 知识库样式必须跟随应用主题变量，不硬编码单一深色或浅色方案。优先使用 `--ui-text-*`、`--ui-surface-*`、`--ui-border-*`、`--ui-button-*`、`--ui-input-*`、`--primary-color` 和 `--background-color`。
- 页面根节点可以为编辑器子组件提供局部别名，但别名必须映射回应用主题变量，不能形成独立主题系统。

## 8. AI Inspector 设计

V1.3 起，右侧 Inspector 的 AI tab 承载知识库内部基础问答，不要求用户离开当前页面。

### 8.1 信息结构

AI tab 采用紧凑工具面板结构：

```text
AI tab
├─ 范围条：当前页面 / 当前空间 / 当前库 / 选区
├─ 助手角色选择
├─ Provider / 模型选择
├─ 快捷动作：总结 / 改写 / 提取 Todo / 标签建议
├─ 问答消息区
├─ 引用来源卡片
└─ 输入区：提问 / 停止 / 重新生成 / 在 AI 页面继续
```

### 8.2 交互规则

- 默认范围是当前页面；如果用户在编辑器中选中文本，优先显示选区范围。
- 空间节点选中时可切换到当前空间问答；库级问答需要用户显式切换。
- 助手角色默认使用 AI 设置中的默认助手；如果助手指定了知识库范围，范围条展示助手范围并允许用户切回当前页面/空间。
- Provider / 模型默认跟随所选助手；助手未指定时使用 AI Chat 默认配置。
- 发送前必须可见地展示将发送的范围、角色、Provider 和模型。
- 回答中的引用以来源卡片展示，卡片提供跳转页面、定位附件或回到来源文档的操作。
- “在 AI 页面继续”使用图标+文字按钮，跳转后保留同一问题、回答、引用和当前知识库范围。

### 8.3 状态

- 未配置 AI：显示配置提示和跳转设置按钮，不报错。
- 未建立索引：允许普通模型问答，但知识库问答按钮 disabled，并提示需要先完成索引。
- 无引用结果：回答仍可展示，但必须标记为“未找到可验证知识库来源”。
- 生成中：输入区显示停止按钮，范围和模型选择暂时锁定。
- 错误：保留用户问题，展示可重试错误，不清空上下文。
